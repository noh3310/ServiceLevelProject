# ServiceLevelProject

## 앱 기능
가까이 있는 친구와 함께 취미를 공유할 수 있는 앱입니다. 회원가입과 로그인을 수행하고, 내가 함께 공유하고 싶은 취미를 등록하고, 나와 취미가 같은 주변 친구를 지도에서 확인하고 친구를 추가 할 수 있습니다. 친구를 맺은 후 채팅을 할 수 있고, 채팅이 끝난 후 친구에 대한 리뷰를 남길 수 있습니다.
